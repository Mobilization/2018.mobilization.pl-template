A) Introduction - top

Eventor is a professional Conference and Event Template. This HTML template specially designed for Conference, Meet-up, Seminars and event related business website. It also comes with twitter bootstrap 3, Video Popup, JCountre, Beautiful Tab, Speakers, Gallery and more.

Template Features

  • 4 Home Variations
  • 2 Different Header Layout
  • Multipage Design
  • HTML5 Validated
  • Based on Twitter Bootstrap 3
  • Responsive Template
  • FontAwesomeicons
  • Isotope
  • Google Web Fonts
  • Contact Form
  • Stylist News Page
  • Documentation
  • Counter
  • Easy to customize and user friendly

H) FAQs - top

How to change Contact Form Email Address for own?

  1. Please, open contact-form.php
  2. Find line 4 you will see
    $recipient = "yourmail@domain.com";
  3. You should change 'yourmail@domain.com' with your own.

How to change Map Address?

  1. With editor please open index.html, venue.html, contact.html etc. files >> go at the bottom and see the below screen shot. Change your Google API key from line # 1206. Here is the link how to get Google MAP API KEY

  2. Then open custom.js file and from line # 50 change Latitude and Longitude. Here is the doc to use latitude and longitude.
  3. After that Change Address from line # 66

How to change Event Date?

  1. Please, open custom.js file >> From line # 39 Change the date of your Event.
